# Session 14 — Text Spacing

در این جلسه خاصیت‌هایی را می‌بینیم که فاصله و رفتار متن را کنترل می‌کنند:

`text-indent`، `line-height`، `letter-spacing`، `word-spacing`، `white-space`

---

## 1. `letter-spacing`
فاصله بین **حروف** را کم یا زیاد می‌کند.

```css
.box h1 {
  letter-spacing: 2px;
}
```

عدد مثبت → حروف از هم بازتر  
عدد منفی → حروف به هم نزدیک‌تر  

مناسب برای عنوان‌ها و تأکید بصری روی کلمات کوتاه.

---

## 2. `word-spacing`
فاصله بین **کلمات** را تنظیم می‌کند.

```css
.box h1 {
  word-spacing: 10px;
}
```

فقط فاصله بین کلمات را عوض می‌کند، نه فاصله حروف داخل یک کلمه را.

---

## 3. `text-indent`
اولین خط پاراگراف را به داخل می‌برد (تورفتگی ابتدای پاراگراف).

```css
.box p {
  text-indent: 20px;
}
```

در این مثال خط اول پاراگراف ۲۰ پیکسل از چپ فاصله می‌گیرد. بقیه خطوط تحت تأثیر این خاصیت نیستند.

---

## 4. `line-height`
فاصله عمودی بین خطوط متن (ارتفاع خط) را مشخص می‌کند.

```css
.box p {
  line-height: 1.4;
}
```

مقدار بدون واحد (مثل `1.4`) معمولاً بهترین انتخاب است، چون نسبت به `font-size` محاسبه می‌شود:

```text
ارتفاع خط ≈ font-size × 1.4
```

مقادیر رایج خوانایی: حدود `1.4` تا `1.8` برای متن بدنه.

می‌توان با `px` یا `rem` هم نوشت، ولی عدد نسبی انعطاف‌پذیرتر است.

---

## 5. `white-space`
رفتار فاصله‌های خالی و شکستن خط را کنترل می‌کند.

| مقدار | معنی |
| --- | --- |
| `normal` | فاصله‌های اضافه جمع می‌شوند؛ متن در صورت نیاز می‌شکند (پیش‌فرض) |
| `nowrap` | متن در یک خط می‌ماند و نمی‌شکند |
| `pre` | مثل `<pre>`؛ فاصله‌ها و خط‌های جدید HTML حفظ می‌شوند |
| `pre-wrap` | فاصله‌ها حفظ می‌شوند و در صورت نیاز می‌شکند |
| `pre-line` | خط‌های جدید حفظ می‌شوند؛ فاصله‌های اضافه جمع می‌شوند |
| `wrap` | اجازه شکستن خط (رفتار نزدیک به حالت عادی در این زمینه) |

```css
.box p {
  white-space: wrap;
}
```

در این جلسه برای پاراگراف `wrap` گذاشته شده تا متن بتواند داخل باکس بشکند و در چند خط دیده شود. اگر `nowrap` بگذارید، متن از باکس بیرون می‌زند.

---

## خلاصه مثال‌های این جلسه

| عنصر | خاصیت | مقدار | اثر |
| --- | --- | --- | --- |
| `h1` | `letter-spacing` | `2px` | فاصله بین حروف |
| `h1` | `word-spacing` | `10px` | فاصله بین کلمات |
| `p` | `text-indent` | `20px` | تورفتگی خط اول |
| `p` | `line-height` | `1.4` | فاصله بین خطوط |
| `p` | `white-space` | `wrap` | اجازه شکستن متن |

> تفاوت سریع: `letter-spacing` = حروف، `word-spacing` = کلمات، `line-height` = خطوط، `text-indent` = فقط خط اول.
