# Session 7 — Margin

`margin` فاصله **بیرونی** عنصر با اطرافش است (بین عنصر و عنصرهای کناری یا لبه‌های صفحه).

در این جلسه چند باکس رنگی داریم تا اثر مقادیر مختلف margin را ببینیم. اول `margin` پیش‌فرض `body` را صفر می‌کنیم تا فاصله اضافه مرورگر مزاحم نشود:

```css
body {
  margin: 0;
}
```

---

## 1. Margin یک سمت
می‌توان فقط یک طرف را تنظیم کرد:

| خاصیت | سمت |
| --- | --- |
| `margin-top` | بالا |
| `margin-right` | راست |
| `margin-bottom` | پایین |
| `margin-left` | چپ |

```css
.box1 {
  margin-bottom: 50px; /* فقط پایین */
}
```

---

## 2. Shorthand با یک مقدار
یک عدد برای هر چهار سمت:

```css
.box2 {
  margin: 50px; /* top right bottom left = 50px */
}
```

---

## 3. Shorthand با دو مقدار
اولی برای بالا/پایین، دومی برای چپ/راست:

```css
.box3 {
  margin: 50px 20px;
  /* بالا و پایین: 50px */
  /* چپ و راست: 20px */
}
```

---

## 4. Shorthand با سه مقدار
ترتیب: بالا → چپ/راست → پایین

```css
.box4 {
  margin: 50px 20px 10px;
  /* بالا: 50px */
  /* چپ و راست: 20px */
  /* پایین: 10px */
}
```

---

## 5. Shorthand با چهار مقدار
ترتیب ساعت‌گرد: **بالا → راست → پایین → چپ**

```css
.box5 {
  margin: 50px 20px 10px 30px;
  /* top: 50px */
  /* right: 20px */
  /* bottom: 10px */
  /* left: 30px */
}
```

---

## خلاصه مثال‌های این جلسه

| کلاس | رنگ | margin |
| --- | --- | --- |
| `.box1` | قرمز | فقط `margin-bottom: 50px` |
| `.box2` | آبی | `50px` برای هر چهار سمت |
| `.box3` | سبز | `50px` عمودی، `20px` افقی |
| `.box4` | زرد | سه مقدار: بالا / چپ‌راست / پایین |
| `.box5` | بنفش | چهار مقدار جدا برای هر سمت |

> نکته: `margin` فاصله بیرون عنصر است؛ فاصله داخل محتوا با `padding` تنظیم می‌شود (جلسه جدا).
