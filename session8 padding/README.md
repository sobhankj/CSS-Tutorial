# Session 8 — Padding

`padding` فاصله **داخلی** عنصر است؛ بین محتوا (متن) و لبه‌های باکس.

تفاوت مهم با جلسه قبل:

| خاصیت | کجا فاصله می‌گذارد |
| --- | --- |
| `margin` | بیرون عنصر |
| `padding` | داخل عنصر، دور محتوا |

مثل جلسه margin، اول `margin` پیش‌فرض `body` را صفر می‌کنیم:

```css
body {
  margin: 0;
}
```

---

## 1. Shorthand با یک مقدار
یک عدد برای هر چهار سمت داخل باکس:

```css
.box1 {
  padding: 20px; /* top right bottom left = 20px */
}
```

متن از هر چهار طرف ۲۰ پیکسل از لبه فاصله می‌گیرد.

---

## 2. Shorthand با دو مقدار
اولی برای بالا/پایین، دومی برای چپ/راست:

```css
.box2 {
  padding: 20px 40px;
  /* بالا و پایین: 20px */
  /* چپ و راست: 40px */
}
```

---

## سایر حالت‌های shorthand (مثل margin)

```css
padding: 20px 40px 10px;       /* بالا | چپ‌راست | پایین */
padding: 20px 40px 10px 30px;  /* بالا | راست | پایین | چپ */
```

یا فقط یک سمت:

```css
padding-top: 20px;
padding-right: 40px;
padding-bottom: 10px;
padding-left: 30px;
```

---

## خلاصه مثال‌های این جلسه

| کلاس | رنگ | padding |
| --- | --- | --- |
| `.box1` | بنفش | `20px` برای هر چهار سمت |
| `.box2` | سبز | `20px` عمودی، `40px` افقی |

`.box1` همچنین `margin-bottom: 20px` دارد تا بین دو باکس فاصله بیرونی ایجاد شود.
