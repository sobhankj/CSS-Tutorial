# Session 11 — Outline

`outline` یک خط دور عنصر است که معمولاً برای نشان دادن **فوکوس** (مثلاً وقتی با Tab روی دکمه یا input می‌روید) استفاده می‌شود.

در این جلسه دو مثال داریم:

```css
.btn {
  border: 3px solid #000000;
  outline: 3px solid #ff0000;
  outline-offset: 10px;
}

.input {
  outline: none;
}
```

---

## تفاوت Outline با Border

| موضوع | `border` | `outline` |
| --- | --- | --- |
| جای رسم | بخشی از box model است | بیرون از box model است |
| روی اندازه باکس | روی `width`/`height` اثر می‌گذارد (مگر `border-box`) | اندازه باکس را عوض نمی‌کند |
| گوشه گرد | با `border-radius` هماهنگ می‌شود | معمولاً گوشه‌ها را مثل border دنبال نمی‌کند |
| فاصله از عنصر | چسبیده به لبه | با `outline-offset` می‌توان فاصله داد |
| سمت جدا | `border-top` و ... دارد | معمولاً یک خط کامل دور عنصر است |
| کاربرد رایج | ظاهر دائمی UI | نشان فوکوس / تأکید موقت |

نکته مهم: `outline` در محاسبات اندازه (`width`، `height`، layout) دخالت نمی‌کند؛ برای همین روی چیدمان صفحه اثر نمی‌گذارد.

---

## خاصیت‌های Outline

مثل border، سه بخش اصلی دارد و shorthand هم دارد:

```css
outline: 3px solid #ff0000;
/* outline-width | outline-style | outline-color */
```

و فاصله از لبه عنصر:

```css
outline-offset: 10px;
```

در `.btn`، outline قرمز ۱۰ پیکسل بیرون از border مشکی رسم می‌شود.

---

## چرا معمولاً `outline: none` می‌گذاریم؟
مرورگرها به‌صورت پیش‌فرض روی عناصر قابل فوکوس (مثل `button` و `input`) outline نشان می‌دهند. گاهی این outline با طراحی UI هم‌خوان نیست؛ برای همین طراح‌ها اغلب می‌نویسند:

```css
.input {
  outline: none;
}
```

یا کلی‌تر:

```css
button:focus,
input:focus {
  outline: none;
}
```

### هشدار دسترس‌پذیری
اگر outline را حذف کنید، کاربر کیبورد ممکن است نداند الان روی کدام عنصر است. پس اگر `outline: none` می‌گذارید، بهتر است جایگزین واضح داشته باشید، مثلاً:

```css
.input:focus {
  outline: none;
  border-color: #0066ff;
  box-shadow: 0 0 0 3px rgba(0, 102, 255, 0.3);
}
```

در پروژه‌های واقعی: outline پیش‌فرض را کورکورانه برای همه حذف نکنید؛ یا جایگزین بگذارید، یا فقط جایی حذف کنید که فوکوس دیگری طراحی شده باشد.

---

## خلاصه مثال‌های این جلسه

| کلاس | چه می‌کند |
| --- | --- |
| `.btn` | هم `border` دارد هم `outline` قرمز با فاصله ۱۰ پیکسل |
| `.input` | `outline: none` تا حلقه فوکوس پیش‌فرض دیده نشود |

با کلیک روی دکمه یا فوکوس روی input، تفاوت رفتار outline را ببینید.
