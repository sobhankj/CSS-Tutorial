# Session 23 — Pseudo-elements (`::before` & `::after`)

Pseudo-element با `::` نوشته می‌شود و یک **عنصر مجازی** قبل یا بعد از محتوای واقعی می‌سازد؛ بدون اینکه تگ جدیدی در HTML اضافه کنید.

دو مورد اصلی:

| Pseudo-element | کجا ساخته می‌شود؟ |
| --- | --- |
| `::before` | قبل از محتوای عنصر |
| `::after` | بعد از محتوای عنصر |

شکل کلی:

```css
selector::before {
  content: '';
  /* بقیه استایل‌ها */
}

selector::after {
  content: '';
}
```

---

## شرط اجباری: `content`
بدون `content` معمولاً چیزی دیده نمی‌شود.

```css
content: '';           /* خالی — برای شکل‌های تزئینی */
content: 'متن';       /* متن مجازی */
content: url('...');   /* تصویر */
```

حتی اگر فقط یک خط یا باکس تزئینی بخواهید، باز هم `content: ''` لازم است.

---

## با `::before` و `::after` چه کارهایی می‌شود کرد؟
تقریباً مثل یک عنصر واقعی استایل می‌گیرند:

- متن اضافه کنار محتوا
- آیکون، نشان، badge
- خط روی قیمت (مثل این جلسه)
- پس‌زمینه تزئینی، کادر، فلش
- افکت hover و دکور UI
- ترکیب با `position` برای قرارگیری دقیق

یعنی خیلی از تزئین‌ها را بدون شلوغ کردن HTML می‌توان با همین دو تا ساخت.

---

## تفاوت با Pseudo-class
| | Pseudo-class | Pseudo-element |
| --- | --- | --- |
| علامت | `:` مثل `:hover` | `::` مثل `::before` |
| معنی | حالت عنصر | ساخت بخش مجازی از عنصر |
| مثال | `:focus`، `:checked` | `::before`، `::after` |

---

## مثال این جلسه
یک قیمت داریم و با `::before` یک خط قرمز روی آن می‌کشیم (مثل قیمت خط‌خورده):

```html
<h1 class="price">189 $</h1>
```

```css
.container {
  position: relative; /* مرجع برای absolute */
}

.price::before {
  content: '';
  display: inline-block;
  position: absolute;
  height: 2px;
  width: 73px;
  background-color: #ff0000;
  top: 0;
  bottom: 0;
  margin: auto; /* وسط‌چین عمودی خط */
}
```

چطور کار می‌کند:
1. `::before` یک عنصر مجازی خالی می‌سازد
2. با `height` و `background` تبدیل به خط قرمز می‌شود
3. با `position: absolute` روی متن قیمت قرار می‌گیرد
4. والد `relative` است تا مختصات خط نسبت به همان باکس باشد

همین کار را می‌توان با `::after` هم کرد؛ فقط عنصر مجازی **بعد** از محتوا ساخته می‌شود.

---

## الگوی رایج

```css
.element {
  position: relative;
}

.element::before,
.element::after {
  content: '';
  position: absolute;
  /* اندازه، رنگ، جایگذاری */
}
```

---

## خلاصه

| موضوع | نکته |
| --- | --- |
| `::before` | قبل از محتوا |
| `::after` | بعد از محتوا |
| `content` | اجباری است |
| کاربرد | تزئین و UI بدون HTML اضافه |
| مثال جلسه | خط قرمز روی قیمت با `::before` |

> نکته: برای یک عنصر معمولاً فقط یک `::before` و یک `::after` دارید؛ اگر به عناصر مجازی بیشتری نیاز باشد، باید تگ واقعی اضافه کنید یا روش دیگری استفاده کنید.
