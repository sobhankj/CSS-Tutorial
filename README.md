# CSS Tutorial — یادداشت‌های یادگیری من

محتوای این تمرین‌ها بر اساس دورهٔ CSS سایت [sabzlearn.ir](https://sabzlearn.ir) بوده است. هر پوشه مربوط به یک جلسه است و معمولاً شامل یک `index.html`، فایل CSS و در خیلی از جلسات یک `README` کوتاه از چیزی که یاد گرفتم می‌شود.

---

## داخل این ریپو چه خبر است؟

یادگیری گام‌به‌گام CSS از مبانی تا انیمیشن و ریسپانسیو. هر `session` یک موضوع کوچک است که با مثال ساده امتحان کرده‌ام.
