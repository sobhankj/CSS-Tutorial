# Session 15 — Text Shadow

`text-shadow` به متن **سایه** می‌دهد و باعث می‌شود برجسته‌تر یا سه‌بعدی‌تر دیده شود.

---

## ساختار `text-shadow`

```css
text-shadow: offset-x  offset-y  blur-radius  color;
```

| بخش | معنی |
| --- | --- |
| `offset-x` | جابه‌جایی افقی سایه (مثبت = راست، منفی = چپ) |
| `offset-y` | جابه‌جایی عمودی سایه (مثبت = پایین، منفی = بالا) |
| `blur-radius` | میزان محو بودن سایه (اختیاری؛ پیش‌فرض `0`) |
| `color` | رنگ سایه |

---

## مثال‌های این جلسه

```css
h1 {
  text-shadow: 5px 5px 2px #bfbfbf;
}

p {
  color: #6738e9;
  text-shadow: 5px 5px 2px #caa6ff;
}
```

یعنی برای هر دو:

- سایه `5px` به راست
- سایه `5px` به پایین
- محو شدن `2px`
- رنگ خاکستری روشن برای `h1` و بنفش روشن برای `p`

---

## چند نکته کاربردی

**۱. بدون blur**  
اگر فقط دو مقدار بنویسید، سایه تیز است:

```css
text-shadow: 3px 3px #000;
```

**۲. چند سایه با هم**  
سایه‌ها را با `,` جدا کنید:

```css
text-shadow: 2px 2px 2px #999, -2px -2px 2px #eee;
```

**۳. خوانایی**  
سایه خیلی قوی یا تیره روی متن کوچک، خواندن را سخت می‌کند. معمولاً برای عنوان‌ها بهتر جواب می‌دهد.

**۴. تفاوت با `box-shadow`**  
`text-shadow` روی حروف متن است؛ `box-shadow` روی کل باکس/عنصر.

---

## خلاصه

| عنصر | `text-shadow` | اثر |
| --- | --- | --- |
| `h1` | `5px 5px 2px #bfbfbf` | سایه خاکستری ملایم |
| `p` | `5px 5px 2px #caa6ff` | سایه بنفش هماهنگ با رنگ متن |
