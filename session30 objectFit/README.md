# Session 30 — Object Fit

`object-fit` مشخص می‌کند **محتوای یک تصویر (یا ویدیو)** چطور داخل باکس خودش جا بگیرد؛ وقتی ابعاد عنصر با نسبت تصویر یکی نیست.

در این جلسه یک `img` داخل یک مربع ۳۰۰×۳۰۰ با `object-fit: cover` دیده می‌شود.

---

## 1. مشکل بدون `object-fit`

اگر تصویر را فقط با `width` و `height` بکشید، ممکن است:

- تصویر کش بیاید و **دفرمه** شود
- یا فقط بخشی از باکس را پر کند و فضای خالی بماند

`object-fit` مثل `background-size` برای تصاویر معمولی عمل می‌کند.

---

## 2. شرط مهم: `width` و `height` روی ۱۰۰٪

برای اینکه `object-fit` درست کار کند، تصویر باید **کل فضای والد را پر کند**. در این جلسه:

```css
.container {
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}
```

| بخش | چرا لازم است |
| --- | --- |
| `.container` با عرض و ارتفاع ثابت | باکس محدود برای تصویر |
| `img { width: 100%; height: 100%; }` | تصویر دقیقاً هم‌اندازه باکس می‌شود |
| `object-fit` | تعیین می‌کند محتوا داخل این باکس چطور برش/کشیده شود |

اگر `width` و `height` را ۱۰۰٪ نگذارید، تصویر ممکن است با اندازه طبیعی خودش نمایش داده شود و `object-fit` اثر واضحی نداشته باشد. پس **اول اندازه عنصر را با باکس یکی کنید، بعد `object-fit` را اعمال کنید.**

---

## 3. مثال این جلسه — `cover`

```css
img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}
```

| بخش | معنی |
| --- | --- |
| `width: 100%` | عرض تصویر = عرض `.container` |
| `height: 100%` | ارتفاع تصویر = ارتفاع `.container` |
| `object-fit: cover` | تصویر باکس را کامل پر می‌کند؛ ممکن است بخشی از لبه‌ها بریده شود |

مربع سیاه ۳۰۰×۳۰۰؛ تصویر داخلش کامل پر می‌شود بدون کش آمدن نامتناسب.

---

## 4. مقادیر رایج `object-fit`

```css
object-fit: fill;       /* کل باکس را پر می‌کند؛ ممکن است کش بیاید */
object-fit: contain;    /* کل تصویر دیده می‌شود؛ ممکن است فضای خالی بماند */
object-fit: cover;      /* باکس کامل پر می‌شود؛ ممکن است برش بخورد */
object-fit: none;       /* اندازه اصلی تصویر */
object-fit: scale-down; /* مثل none یا contain — هر کدام کوچک‌تر باشد */
```

| مقدار | رفتار |
| --- | --- |
| `fill` | پر کردن باکس، حتی با دفرمه شدن |
| `contain` | حفظ نسبت؛ تمام تصویر دیده می‌شود |
| `cover` | حفظ نسبت؛ باکس کامل پر می‌شود |
| `none` | بدون مقیاس‌دهی اجباری |
| `scale-down` | کوچک‌تر از `none` و `contain` |

---

## 5. `object-position`

با `cover` یا `contain` می‌توان مرکز برش را عوض کرد:

```css
object-fit: cover;
object-position: center;     /* پیش‌فرض */
object-position: top;
object-position: left top;
object-position: 20% 30%;
```

---

## 6. روی چه عناصری کار می‌کند؟

عمدتاً روی عناصر **replaced** مثل:

- `img`
- `video`
- `iframe` (در بعضی موارد)

روی `div` معمولی معمولاً کار نمی‌کند؛ برای پس‌زمینه از `background-size` استفاده کنید.

---

## خلاصه مثال این جلسه

| بخش | مقدار | نتیجه |
| --- | --- | --- |
| `.container` | `300px × 300px` | باکس ثابت |
| `img` | `width: 100%` + `height: 100%` | پر کردن کامل باکس |
| `img` | `object-fit: cover` | پوشش بدون دفرمه شدن (با برش احتمالی) |

یک تصویر داخل مربع ۳۰۰×۳۰۰؛ به‌خاطر `width`/`height` صددرصد و `cover`، باکس کامل پر می‌شود و نسبت تصویر حفظ می‌ماند.
